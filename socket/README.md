# Google Docs Clone
This repository consists of socket code for real time update propagation in quill rich text editor.
The following modules are included in this application:
<ol>
    <li><strong>Socket.IO: </strong>This is a javascript module which incorporate socket connection between different hosts.</li>
    <li><strong>Mongoose: </strong>It is a javascript module which is used to connect with mongo DB, which is used as database for the clone.</li>
</ol>
